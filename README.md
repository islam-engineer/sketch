# Sketches Collection Repository

A curated collection of drawing sketches organized into multiple categories. This repository contains sketch assets, text art, and supporting files that can be used for learning, inspiration, educational projects, or creative applications.

## 📂 Repository Structure

```
.
├── Animal/
├── DC/
├── Games/
├── Onepiece/
├── Sports/
├── Textart/
├── Transportation/
│   └── Cars/
├── kidscartoon/
├── UpdateSketchesJson.json
├── MusnadImamAhmadBook.pdf
└── README.md
```

## 📁 Categories

### 🐾 Animal
Collection of animal sketches.

### 🦸 DC
DC Comics character sketches.

### 🎮 Games
Sketches inspired by popular video games.

### ☠️ One Piece
One Piece anime and manga character sketches.

### 🏅 Sports
Sports-related sketches and illustrations.

### ✍️ Textart
Creative text-based artwork and designs.

### 🚗 Transportation / Cars
Vehicle and automobile sketches.

### 🧒 Kids Cartoon
Cartoon sketches suitable for children.

## 📄 Files

### UpdateSketchesJson.json
Contains metadata and updates for the sketch collection. This file can be used by applications to dynamically load or manage available sketches.

### MusnadImamAhmadBook.pdf
A PDF document included in the repository.

## ✨ Features

- Well-organized category structure
- Easy to browse
- JSON metadata support
- Suitable for educational and creative projects
- Continuously updated with new sketches

## 🚀 Usage

Clone the repository:

```bash
git clone https://github.com/islam-engineer/sketch.git
```

Browse the folders to access the sketches you need.

If your application uses `UpdateSketchesJson.json`, load it to retrieve the available sketch information programmatically.

## 🤝 Contributing

Contributions are welcome.

If you'd like to add new sketches:

1. Fork the repository.
2. Create a new branch.
3. Add your sketches to the appropriate category.
4. Update `UpdateSketchesJson.json` if necessary.
5. Submit a Pull Request.

This repository is actively maintained and new sketches are added periodically.
